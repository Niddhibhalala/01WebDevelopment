Q.2 How to Apply CSS using JQuery, How to add class and remove class in JQuery, JQuery Animation

Ans. Apply CSS using JQuery has several methods for CSS manipulation
The methods are there:

  • AddClass() : Adds one or more classes to the selected elements.
    For E.g. .addClass(classname, [duration], [easing], [callback])
  • RemoveClass() : Remove one or more classes form the selected elements
    For E.g. .removeClass(ClassName, [duration], [easing], [callback])
  • Toggle class() : Toggle between adding/removing classes form the selected elements
