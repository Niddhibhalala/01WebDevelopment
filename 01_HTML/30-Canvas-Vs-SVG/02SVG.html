SVG:The Scalable vector graphic(SVG) is an XML-based image format that is ussed to define two-dimensional vector-based graphics for th eweb. unlike raster image(Ex; .jpg, .gif, .png, etc),a vector image can be scaled up pr down to any extent without losing the image qaulity

An SVG image is drawn out using a series of statements that follows the xml schema-that means SVG-images can be created and edited with any text editor, sucj as Notepad.There are several other advantages of using SVG over other image formats like .JPEG, .GIF, .PNG, etc.
