Absolute path

what is an absolut path?An absolute path is defined as the specifying the location of a file or directory from the root directory(/). In other words we can say absolute path is a complete path from start of actual filesystem from / directory.

Relative path

what is a relative path?A relative path describes the location of a file relative to the current working directory.
