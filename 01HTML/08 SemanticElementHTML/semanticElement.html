  • <mark>
    The <⁢mark> element is found within the <⁢body> tag in HTML.It is used to hightlight some part of the texr for quick reference inside another element,such as in a paragraph that is of more importance or relevance to the user.It uses the default color yellow for highlighting.This can be changed using CSS properties.

  • <detail>
    This tag specifies additional details that the user can open and close on demand.

  • <Summary>
    It defines a visible heading for the <⁢details> element.The <⁢summary> element should be the first child element of the <⁢detail>element.

  • <figcaption>
    The <⁢figcaption> tag defines a caption for a <⁢figure> element.The <⁢figcaption> element can be placed as the frist or last child of the <⁢figure> element.

  • <figure>
    This tag specifies self-contained content.like illustations, diagrams, photos, code listings, etc.It should not affect the flow of the document.

  • <article>
    The <⁢article> HTML element represents a self-contained composition in a document,page,application, or site, which is intended to be independently distributable or reusable.

  • <aside>
    It defines some content aside from the content it is placed in.

  • <footer>
    The <⁢footer> tag in HTML5 is used to define a footer of HTML element.

  • <header>
    The <⁢header> element represents a container for introductory content or a set of navigational links.A <⁢header> element typically contains: one or more heading element h1 to h6.

  • <main>
    The <⁢main> tag specifies the main content of a document.

  • <nav>
    The <⁢nav> HTML element represents a section of a page whose purpose is to provide navigation links,either within the current document or to other documents.

  • <section>
    <⁢section> tag defines the section of documents such as chapters,headers,footers or any other sections.It divides the content into section and subsections.

  • <time>
    The <⁢time> tag defines a specific Time or Datetime.
